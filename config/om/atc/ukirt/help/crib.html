ORAC Observing Crib Sheet

This page gives a quick summary of the steps involved in running ORAC to take data at the summit. Read the step-by-step guides for a more detailed explanation of how to use it. To start ORAC for observing, log into kiki and type the commands
orac_ufti_new
dramastart
oracom
to start up the ORAC Observing Manager (ORAC-OM).

Go into the computer room and reboot the UFTI crate

A login in screen will appear. Enter your user-id and password into the appropriate boxes and click on OK.

A screen which tracks user information for UKIRT operations and for the datafile headers will the appear. You MUST fill out the entry for observer names and your PATT run number before the software will allow you to proceed. Click on OK.

The main Observing Manager screen will now startup.

Do not carry out any further steps until you see a message saying "connection info" on the x-term you started from.

On the left you will see a list of your science programs which you have submitted to the database. Move the highlight to the program you want to use, then click on Fetch to fetch the program from the database to the ORAC-OM.

When your program has been fetched from the database a listing of all the Observations you have created within it will appear in the "Fetched Science Program" area on the right. The order in which the observations appear in the list is the same as they were in the Science Program when you created it using the OT.

Move the highlight to the observation that you actually want to be executed and click on send for execution . Note that the action of this button is only to start up the sequencer console and translate your Observaiton into executable form. The system will not start to take data until you tell it to do so via the sequencer console.

It is helpful to organise the main Observing Manager window and the Observation Sequencer Console on the screen so that you can see both at the same time. (When you are ready to use a new Observation you will do so from the Observing Manager).

To start data taking press the Run button on the sequencer console.

To stop the sequence before it reaches the end use either the Stop asap button or the stop at break button on the sequencer console.

To start a sequence from any position, move the highlight to any item and then use the Run From Highlight < /font > button.

To start the next Observation when the current one is finished : move the highlight on the main ORAC-OM screen to the one you want to do, and click on send for execution . It will be sent to the sequencer and replaces the old one. Use Run to start it.

To Exit from the system

Exit any console windows using File/Exit
Exit the main oracom window by using the Exit OM buttom
Exit the Quick Look using File/Exit
